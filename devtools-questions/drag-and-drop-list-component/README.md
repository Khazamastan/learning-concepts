# Drag and Drop List Component

Simple React component that uses native HTML drag-and-drop events to reorder a vertical list.

## Features

- Drag to reorder items; the list updates immediately.
- Visual feedback for the dragged item.
- Reset button returns to the initial order.

## Running locally

```bash
cd drag-and-drop-list-component
npm install
npm run dev
```

Visit `http://localhost:5173`.
