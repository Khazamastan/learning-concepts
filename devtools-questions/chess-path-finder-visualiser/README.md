# Chess Path Finder Visualiser

Interactive React demo that shows the shortest knight path between two squares on an 8×8 chessboard. Uses breadth-first search to compute the path and highlights each move.

## Features

- Click a square to set the destination; Shift + click to set the starting square.
- BFS finds the optimal path and lists coordinates in algebraic notation.
- Coloured board cells show start, end, and squares along the path.

## Running locally

```bash
cd chess-path-finder-visualiser
npm install
npm run dev
```

Navigate to `http://localhost:5173`.
