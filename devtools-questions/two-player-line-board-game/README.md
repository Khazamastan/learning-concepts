# Two-Player Line Board Game

Machine-coding solution for a 12×12 dots-and-line board where players alternate turns. First to complete five markers in a row (horizontal, vertical, or diagonal) wins.

## Features

- Highlights the winning line of five markers.
- Move history with ability to jump to any previous state.
- Toggle to reverse move order.
- Supports players Red (Player One) and Green (Player Two).

## Run locally

```bash
cd two-player-line-board-game
npm install
npm run dev
```

Open `http://localhost:5173`.
