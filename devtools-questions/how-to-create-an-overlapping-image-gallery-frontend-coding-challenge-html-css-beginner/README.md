# Overlapping Image Gallery

## Problem

Build a simple overlapping gallery where cards stack with intentional rotation and drop shadow to create depth, while remaining responsive on small screens.

## Solution

Pure HTML/CSS implementation. A wrapping `.gallery` positions three `figure` cards absolutely for desktop and switches to a stacked layout below 720px. Each card uses border radius, shadows, and transforms to achieve the overlap. Hover/focus states lift cards slightly for interactivity.

## Running locally

Open `index.html` in any modern browser.
