# What Is the Paradigm of JavaScript?

## Problem
Summarise the programming paradigms JavaScript supports and provide short code
snippets that highlight the flexibility of the language.

## Solution
The example processes the same array three ways:

- An **imperative** loop mutates an accumulator.
- A **functional** pipeline composes `map`, `filter`, and `reduce`.
- A **prototype-based object** models a counter with shared behaviour.

Running the script prints the result of each approach, underscoring that
JavaScript is a prototype-based, multi-paradigm language capable of imperative,
functional, and object-oriented styles.
