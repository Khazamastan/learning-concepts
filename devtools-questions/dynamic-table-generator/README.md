# Dynamic Table Generator

React component that lets you add/remove columns and rows on the fly. Cells are editable and removing a column updates every row.

## Features

- Create columns by name (slugified for keys).
- Append rows with blank cells.
- Inline editing for each cell.
- Remove columns with one click.

## Run locally

```bash
cd dynamic-table-generator
npm install
npm run dev
```

Navigate to `http://localhost:5173`.
